Slowloris Attack

About

Slowloris is a type of denial-of-service attack designed to keep a web server connection open and hold it indefinitely. It achieves this by sending partial HTTP requests to the target server and keeping the connections open by sending occasional incomplete request headers.

Impact

Slowloris attacks can exhaust a web server's resources, leading to a significant decrease in its ability to serve legitimate requests and causing a denial-of-service condition.

Mitigation

To defend against Slowloris attacks, web servers can implement various countermeasures such as setting connection timeouts, limiting the number of concurrent connections, and deploying intrusion detection systems.
